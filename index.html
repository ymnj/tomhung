About me

Welcome. My name is Tom and I am a Vancouver based web developer. I first started coding a year ago when I decided to teach myself the fundamentals of HTML + CSS + Javascript. After experiencing the joy of creating simple interactive web pages, I was hooked on learning more. Now, as a recent graduate of CodeCore Bootcamp, I have expanded my knowledge of web technologies and feel comfortable building full web applications.

I am currently in the process of learning the MEAN stack and will be using this portfolio page as my project so things may break occasionally. Please check out my Github to see some of the projects I have worked on or feel free to get in contact with me!
